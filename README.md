Valorant Profiles
=================

This is a web application that allows users to manage their Valorant profiles.

## Features

- View and edit player profiles
- Claim and unclaim player profiles
- View and edit player riot information
- View and edit profile data

## Installation

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Create a virtual environment and activate it.
4. Install the required dependencies using pip.
5. Start building using the following command:

```
python manage.py makemigrations
python manage.py migrate
python manage.py setup_project
python manage.py runserver
```
